

# Novamind

NovaMind es la página de inicio de una plataforma web pensada para cursos en línea, dedicada especialmente a niños y adultos mayores interesados en conocer las tecnologías emergentes y lo nuevo del mundo digital.

## ¿Qué hace esta página?
- Presenta el concepto de NovaMind y su misión educativa.
- Muestra testimonios de usuarios de diferentes edades sobre cómo la tecnología puede transformar sus vidas.
- Incluye una experiencia interactiva inicial, como un juego de memoria, para que los visitantes se diviertan y se familiaricen con la plataforma.
- Ofrece enlaces para registrarse y comenzar el proceso de aprendizaje (aún no hay cursos funcionales implementados).

## Experiencia en la página de inicio
Al ingresar, verás una interfaz colorida, accesible y amigable, con mensajes motivadores, testimonios reales y un minijuego para interactuar. Los botones de registro e inicio de sesión están visibles para que los usuarios puedan comenzar su experiencia.


## Instalación
1. Clona el repositorio:
   ```
   git clone https://github.com/garciatarelo/novamind.git
   ```
2. Coloca la carpeta en tu servidor local (XAMPP, WAMP, etc.).
3. (Opcional) Si quieres conectar con una base de datos, importa `bd/novamind.sql` y configura `php/conexion.php`.


## Cómo probarlo
1. Abre tu navegador y accede a `http://localhost/novamind/`.
2. Explora la página de inicio, lee los testimonios y juega el memorama.
3. Usa los botones para registrarte o iniciar sesión (el registro y login pueden requerir configuración adicional en el backend).


## Tecnologías utilizadas
- HTML, CSS, JavaScript
